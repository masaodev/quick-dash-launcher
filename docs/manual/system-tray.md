# システムトレイメニュー

QuickDashLauncherは、Windowsシステムトレイ（タスクバー右下の通知領域）にアイコンを常駐させ、右クリックメニューから各種機能にアクセスできます。

## メニュー項目

システムトレイアイコンを右クリックすると、以下のメニューが表示されます：

### 1. バージョン情報

```
QuickDashLauncher v0.2.8
```

- 現在実行中のアプリケーションバージョンを表示
- 選択不可（情報表示のみ）
- バージョン番号は `package.json` から自動取得

### 2. 表示

```
表示 (Alt+Space)
```

- メインウィンドウを表示してフォーカスを当てる
- 括弧内には現在設定されているホットキーが表示される
- ホットキーが未設定の場合は `表示` のみ表示

**動作**:
- メインウィンドウが非表示の場合 → 表示してフォーカス
- メインウィンドウが最小化されている場合 → 復元してフォーカス

### 3. 設定...

```
設定...
```

- 管理ウィンドウの「設定」タブを開く
- ホットキー、ウィンドウサイズ、データフォルダパスなどの設定を変更できる

**動作**:
- 管理ウィンドウを開き、「設定」タブをアクティブにする
- 管理ウィンドウが既に開いている場合は、設定タブに切り替える

### 4. データフォルダを開く

```
データフォルダを開く
```

- QuickDashLauncherの設定・データが保存されているフォルダをエクスプローラーで開く
- デフォルト: `%APPDATA%\QuickDashLauncher`
- カスタム設定の場合は設定されたフォルダを開く

**用途**:
- `data.txt`、`data2.txt`、`tempdata.txt` をテキストエディタで直接編集
- `icons/` フォルダ内のアイコンを確認
- 設定ファイルのバックアップ作成

### 5. ヘルプ

```
ヘルプ
```

- QuickDashLauncherのGitHubリポジトリをブラウザで開く
- URL: `https://github.com/masaodev/quick-dash-launcher`

**内容**:
- README（機能説明、使い方）
- リリースノート
- Issues（バグ報告、機能リクエスト）
- ドキュメント

### 6. 終了

```
終了
```

- QuickDashLauncherアプリケーションを完全に終了する
- すべてのウィンドウを閉じ、システムトレイアイコンも削除される

**注意**:
- 終了すると、ホットキーも無効になります
- 再度使用するには、スタートメニューから起動してください

## その他の操作

### ダブルクリック

システムトレイアイコンをダブルクリックすると、メインウィンドウが表示されます。

**動作**:
- 右クリックメニューの「表示」と同じ動作
- メインウィンドウを素早く表示する簡単な方法

## 使用例

### データファイルを直接編集したい

1. システムトレイアイコンを右クリック
2. 「データフォルダを開く」を選択
3. エクスプローラーで `data.txt` をお好みのテキストエディタで開く
4. 編集後、メインウィンドウで「🔄 データ再読み込み」を実行

### ホットキーが効かないときに確認

1. システムトレイアイコンを右クリック
2. 「表示 (Alt+Space)」の括弧内を確認
3. ホットキーが表示されない場合は、「設定...」から再設定

### アプリケーションを完全に終了したい

1. システムトレイアイコンを右クリック
2. 「終了」を選択
3. アプリケーションが終了し、システムトレイアイコンも消える
