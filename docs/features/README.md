# 横断的機能ドキュメント

このフォルダには、**複数の画面にまたがる横断的な機能や概念**についてのドキュメントを配置します。

## このフォルダの役割

### 📋 配置基準

以下の条件に当てはまる機能・概念は、このフォルダにドキュメントを作成します：

1. **複数画面で使用される機能**
   - 例：アイコンシステム（管理画面、登録モーダル、メインウィンドウ等で使用）

2. **全画面に共通する機能**
   - 例：キーボードショートカット（全画面で有効）

3. **概念的な説明が必要な機能**
   - 例：グループ起動の概念、ワークスペースの仕組み

4. **複数のコンポーネントを組み合わせた機能**
   - 例：ワークスペース機能（メインウィンドウ + ワークスペースウィンドウ + システムトレイ）

### ❌ 配置しないもの

以下は **screens/** フォルダの各画面仕様書に記載します：

- 特定の画面でのみ使用される機能
- 画面固有のUI操作手順
- 単一画面内で完結する機能

## 参照方向

```
画面仕様（screens/）
    ↓ 参照
横断的機能（features/）← このフォルダ
    ↓ 参照
技術実装（architecture/）
```

### 使い方

1. **画面仕様から参照する** - 各画面のドキュメント（screens/）から、関連する横断的機能へリンク
2. **概念の説明** - 画面固有の操作ではなく、機能の概念や仕組みを説明
3. **技術詳細へ橋渡し** - 必要に応じて architecture/ の技術ドキュメントへリンク
