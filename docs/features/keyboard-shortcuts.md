# キーボードショートカット

QuickDashLauncherで使用可能なキーボードショートカット一覧です。

## グローバルショートカット

| キー | 動作 |
|------|------|
| `Alt+Space` | ランチャーウィンドウの表示/非表示切り替え（デフォルト、設定で変更可能） |
| `Ctrl+Alt+W` | ワークスペースウィンドウの表示/非表示切り替え |

> **注意**: グローバルホットキー（Alt+Space）は設定画面でカスタマイズできます。他のアプリケーションとの競合を避けるため、お好みのキー組み合わせに変更してください。

## ランチャーウィンドウ内

### 基本操作

| キー | 動作 |
|------|------|
| `Enter` | 選択中のアイテムを起動 |
| `Shift+Enter` | 選択中のアイテムの親フォルダを開く |
| `Escape` | ウィンドウを非表示（一部の状態では無効） |
| `↑` `↓` | アイテム選択の移動 |

**注意**: Escapeキーは、以下の状態ではウィンドウを閉じません：
- 初回起動モード（ホットキー未設定）
- 編集モード（モーダルが開いている）
- モーダルモード（ダイアログ表示中）
- ピン留めモードが`alwaysOnTop`または`stayVisible`

### マウス操作

| 操作 | 動作 |
|------|------|
| 左クリック | アイテムを選択して起動 |
| 右クリック | コンテキストメニューを表示 |
| `Escape` | コンテキストメニューを閉じる |
| メニュー外クリック | コンテキストメニューを閉じる |

### 検索とフィルタリング

| キー | 動作 |
|------|------|
| 文字入力 | リアルタイム検索（スペース区切りでAND検索） |
| `Ctrl+A` | 検索ボックスの全選択 |
| `Ctrl+↑` | 検索履歴を遡る |
| `Ctrl+↓` | 検索履歴を進む |

### タブ操作

| キー | 動作 |
|------|------|
| `Tab` | 次のタブへ移動（タブ表示有効時、複数タブ存在時のみ） |
| `Shift+Tab` | 前のタブへ移動（タブ表示有効時、複数タブ存在時のみ） |

### モード切り替え

| キー | 動作 |
|------|------|
| `Ctrl+E` | アイテム管理の開始/終了 |
| `Ctrl+W` | ワークスペースウィンドウを開く/閉じる（トグル） |

## 管理ウィンドウ

### データ編集

| キー | 動作 |
|------|------|
| `Ctrl+E` | 管理ウィンドウを開く/閉じる（トグル） |
| `Ctrl+W` | ワークスペースウィンドウを開く/閉じる（トグル） |
| `Ctrl+S` | データを保存（アイテム管理タブ） |
| `Delete` | 選択した行を削除（アイテム管理タブ） |
| `Enter` | セル編集の確定 |
| `Escape` | セル編集のキャンセル |

**注意**: 管理ウィンドウを閉じるには、以下のいずれかの方法を使用してください：
- ウィンドウ右上の閉じるボタン（×）
- Ctrl+Eキー（トグル）
- 設定メニューから「閉じる」

**重要**: Escapeキーで管理ウィンドウを閉じることはできません（誤操作防止のため）。

### 開発者向け

| キー | 動作 |
|------|------|
| `Ctrl+Shift+I` | 開発者ツール（DevTools）を開く/閉じる（開発モードのみ） |

### テーブル操作

| 操作 | 動作 |
|------|------|
| クリック | セル編集の開始 |

## システム操作

### アイコン取得

| 操作 | 動作 |
|------|------|
| 🔄ドロップダウン → 🎨 アイコン取得（現在のタブ） | 現在表示中のタブのアイコンが未設定のアイテムのみを対象に一括取得（フォルダとグループを除く） |
| 🔄ドロップダウン → 🎨 アイコン取得（全タブ） | すべてのタブのアイコンが未設定のアイテムを対象に一括取得（フォルダとグループを除く） |

### ウィンドウ制御

| 操作 | 動作 |
|------|------|
| 📌ボタンクリック | ウィンドウ固定/固定解除 |
| システムトレイダブルクリック | ウィンドウ表示 |
| システムトレイ右クリック | システムトレイメニューを表示 |

## 設定メニュー（⚙ボタン）

| 項目 | 動作 |
|------|------|
| ⚙️ 基本設定 | 管理ウィンドウの設定タブを開く（ホットキー、ウィンドウサイズ等） |
| ✏️ アイテム管理 | 管理ウィンドウの編集タブを開く（data.txt編集） |
| 🚪 アプリを終了 | アプリケーションを終了 |

## システムトレイメニュー（右クリック）

| 項目 | 動作 |
|------|------|
| バージョン情報 | 現在のバージョンを表示（選択不可） |
| 表示 (ホットキー) | メインウィンドウを表示（設定されたホットキーも表示） |
| 画面中央に表示 | 設定に関係なく強制的に画面中央に表示 |
| 設定... | 管理ウィンドウの設定タブを開く |
| データフォルダを開く | データフォルダをエクスプローラーで開く |
| ヘルプ | GitHubリポジトリをブラウザで開く |
| 終了 | アプリケーションを終了 |

## ワークスペースウィンドウ

### 基本操作

| キー | 動作 |
|------|------|
| `Ctrl+Alt+W` | ワークスペースウィンドウの表示/非表示切り替え（グローバル） |
| `Ctrl+V` | クリップボードからファイル・フォルダ・URLをペーストして追加（v0.5.1以降） |
| `Enter` | 編集モードで名前を確定 |
| `Escape` | 編集モードをキャンセル |

### マウス操作

| 操作 | 動作 |
|------|------|
| アイテムをクリック | アイテムを起動 |
| アイテムをダブルクリック | アイテム名の編集モードに移行 |
| アイテムを右クリック | コンテキストメニューを表示 |
| アイテムをドラッグ | アイテムの並び替え・グループ間移動 |
| グループヘッダーをドラッグ | グループの並び替え |

## ダイアログ操作

### AlertDialog（通知ダイアログ）

| キー | 動作 |
|------|------|
| `Escape` | ダイアログを閉じる |
| `Enter` | ダイアログを閉じる（OKボタンと同等） |

### ConfirmDialog（確認ダイアログ）

| キー | 動作 |
|------|------|
| `Escape` | ダイアログをキャンセルして閉じる |
| `Enter` | 確認を実行してダイアログを閉じる |

### FilePickerDialog（ファイル選択ダイアログ）

| キー | 動作 |
|------|------|
| `Escape` | ダイアログをキャンセルして閉じる |
